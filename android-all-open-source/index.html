概述

此博客初衷是记录一下自己开发过程后使用到或者收集到的一些轮子和开源项目，不追求全，只追求实用吧。会不断的更新,也是希望能继续完善吧。
推荐
Github最全面的Android开源项目汇总

Loading

  • MetaballLoading
    水滴传递的加载效果
    github:https://github.com/dodola/MetaballLoading
  • WhorlView
    漩涡风格的加载效果
    github:https://github.com/Kyson/WhorlView
  • AVLoadingIndicatorView
    Android Loading动画的一个集合
    github:https://github.com/81813780/AVLoadingIndicatorView
  • shapeLoadingView
    仿58loading效果
    github:https://github.com/zzz40500/android-shapeLoadingView

ViewPager

  • SpringIndicator
    viewapger指示器，像水滴流动的效果

    • ProductTour
      ViewPager的PageTransformer切换动画
    • VerticalViewawPager
      VerticalVie向上滑动的viewpager,没有衔接效果

    指示器

    • FlycoTabLayout
      包含两种TabLayout.[SlidingTabLayout是基于PagerSlidingTabStrip的大量修改，新增了很多自定义的功能。CommonTabLayout:不同于SlidingTabLayout对ViewPager依赖,它是一个不依赖ViewPager可以与其他控件自由搭配使用的TabLayout.

    Dialog

    • FlycoDialog
      多种样式与动画效果的对话框

ListView

  • 瀑布流
  • IndexView
    索引控件可拖动，可根据列表滚动移动
  • SwipeMenuListView(右滑删除)

侧滑菜单

  • DragLayout
    使用 support.v4 包下的 ViewDragHelper 实现 QQ5.0 侧滑
  • FlowingDrawer()
    具有弹性的侧滑菜单

下拉上拉刷新

  • BGARefreshLayout-Android
    多种下拉刷新效果(支持上拉刷新)
  • UltimateRecyclerView
    支持ListView、RecycleView上拉刷新以及右滑动删除，支持拖动，动画，和和粘性头部。隐藏和显示Actionbar和ToolBar等

  • Ultra-Pull-To-Refresh
    此库是单纯的下拉刷新的组件,使用起来简单，API LEVEL >= 8可以ui样式十分的简单。还有md风格的下拉刷新样式哦。

    MD

    • MaterialDialog
      MD风格的dialog
    • material library
      Md风格组件的集合
    • RippleView
      兼容低版本的水波纹控件
    • MaterialFilePicker
      MD风格的文件选择器

    效果实现

    • SlidingCard
      相册的效果滑动
    • RadarScanView
      雷达扫描的效果

多媒体

  • AndroidFFmpeg
    Android FFmpeg.用于复杂视频处理(转码、压缩、录制视 频等)
  • android-gpuimage-plus
    Android-gpuimag的增强版。
  • MagicCamera
    集成了美颜滤镜效果。

动画

  • SwitchLayout
    视图切换动画
  • NiftyDialogEffects
    支持自定义飞入动画样式的 Dialog
  • AndroidViewAnimations
    View动画的Android

图像处理

  • ImageBlurring
    图像模糊
  • BitmapMerger
    图片合成库

功能应用

  • pinyin4j
    汉字转拼音,获取汉字首字母等操作。
  • MutiPhotoChoser
    自定义相册多图选择
  • Colorful
    实现了换肤的功能。
  • Apk统计自身被卸载
  • Emoji表情的实现
